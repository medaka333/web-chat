HTMLのformの説明

        
      1 <form role="form" action="/home/search_friend" method="POST">
      2  <div class="form-group">
      3         <input type="text" name="searchBar" id="searchBar" placeholder="検索バー">
      4         <button class="btn btn-lg btn-primary pull-right" type="submit">検索</button>
      5     </div>
      6 </form>
  • 送信ボタンを押すと、form が発火する
  • action には、サーバーにデータを送信する際のパスを書く
  • method には、データの送信方法を書く。主に、GET と POST がある。
  • 3 行目で、ユーザーが入れた情報を name=”名前”で名前を指定してサーバーに送れる。
  • GET はデータを URL に含めて送信し、サーバーからのデータの取得が主である。URL にデータを含めるので、見られても良い情報に限った方が良い。
  • POST はデータをリクエストボディに含めて送信し、サーバーにデータを送信するのが主である
